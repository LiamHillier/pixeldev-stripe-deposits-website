## Installing via WordPress Admin

The easiest way to install the plugin:

1. Navigate to **Plugins > Add New** in your WordPress admin
2. Click **Upload Plugin** at the top of the page
3. Select the `pixeldev-stripe-deposits.zip` file
4. Click **Install Now**
5. After installation completes, click **Activate Plugin**

## Installing via FTP

For manual installation:

1. Extract the `pixeldev-stripe-deposits.zip` file on your computer
2. Connect to your server via FTP
3. Upload the `pixeldev-stripe-deposits` folder to `/wp-content/plugins/`
4. Navigate to **Plugins** in your WordPress admin
5. Find "Deposits and Payment Plans for Stripe WooCommerce" and click **Activate**

## After Activation

Once the plugin is activated:

1. Navigate to **WooCommerce > Settings > Payments**
2. Find **Stripe Deposits** in the list
3. Click **Manage** to access the plugin settings

You'll see the main dashboard with tabs for:
- **Plans** - Create and manage payment plans
- **Active Plans** - View customer payment plans
- **Email Settings** - Configure payment reminders
- **License** - Activate your license key
